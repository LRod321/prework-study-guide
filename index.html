HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML uses the 'id' attribute as a way to identify elements and select them.
  • It is composed of elements defined by tags, like < h1 >, < p >, < a >, etc.
  • Common Tags:
      •< h1 > to < h6 >: Headings.
      •< p >: Paragraph.
      •< a >: Anchor (links).
      •< img >: Image.
      •< ul >, < ol >, < li >: Lists (unordered, ordered, list item).
      •< div >: Division (block-level container).
      •< span>: Inline container.
  • HTML (HyperText markup Language): The standard language for creating webpages.

CSS

  • Cascading Style Sheets(CSS): Used by developers to define how webpage content is presented to users.
  • CSS defines the colors, aesthetiics, visual layout, and organization.
  • inline CSS: Used when a developer wants to style an element directly in the HTML file. useful for quick and permanent changes but are less flexible than external and internal style sheets, becasue each inline style must be edited individually when you make a design. Applies to a specific HTML tag.
  • internal CSS style sheet:Possible to embed entire stylesheet directly in the HTML file if within the < head > element. Used to style multiple elements in the page code
  • external CSS style sheet: Is the most common way to style a webpage. Allows developers to keep all their CSS rules in a seperate file, which makes design changes easier.
  • Declarations contain two important components: The CSS property we want to apply and the value of the property.
  • Margin: Property indicates how much space we want around the outside of an element.
  • Padding: Property adds space around the content inside an element.
  • Special wild card symbol (*): We have declared that ALL of the elemnts on the page will have the style properties (margin, padding) set as 0.
  • Assigning colors in CSS can be done by semantic references such as 'gray' or 'blue'. We can also refer to a color by its hexadecimal code, or hex#
  • The different types of selectors: ID, Class, Attribute and Pseudo-class selector.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull: Fetch and merge changes from the remote repository
  • git push: Push committed changes to the remote repository
  • git merge (branch): Merge a branch into the current branch
  • git status: Check the status of your working directory and staging area
  • git log: Show the commit history
  • git commit -m "message": Commit staged changes with a descriptive message
  • git add (file): Stage changes (prepare them to be committed)
  • git clone (url): Clone an existing repository from a URL
  • Git is used for version control, a system that manages changes to a projects code, documentation, or other digital content over time
  • Allows collaboration between developers to work on different parts of the project simultaneously

JavaScript

  • Can power websites with effects, interactivity, data communication, and more.
  • Nan("Not a Number") is the only value in JavaScript that is not equal to itself.
  • BigInt type is a numeric primitive in JavaScript that can represent intergers with arbitrary magnitude.
  • String type represents extual data and is encoded as a sequence of 16-bit unsigned integer values representing UTF-16 code units
  • Symbol is a unique and immutable primitive value and may be used as the key of an Object property
  • Variable ("var"): simply a named container that allows us to store data in our code.
  • (console.log.) The word console refers to a test environment that developers use to check out their code. The second part, .log(), is a method, which means a set of instructions that can be executed by a computer. The .log() method will output whatever we add inside the parentheses to the console.
  • Control flow is the order in which a computer executes code in a script.
  • A value is considered "falsy" if it is one of the following values: 0, -0, On, "", null, undefined, NaN, or a Boolean data type with the value of false. All other values, including a Boolean with the value of true, are considered "truthy"
  • Using "if" statements allows us to interrupt control flow.
  • Array: A single variable that is used to hold a group of data. Arrays are typically used to hold data that is related in some way.
  • "var": Variable
  • A "for" loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over.
  • Function: Are reusable blocks of code that contain instructions to perform a specific task.
